## Weekly flow

1. Check the semester Project and sort by due date.
2. Jump into the class repo.
3. Work inside homework or labs.
4. Commit and push at the end of each work session.
5. Copy final deliverables into the submission folder.

## Repo map

templates: reusable markdown, latex, and writeup scaffolds  
scripts: small helper scripts you might reuse across classes  
docs: optional, for any semester wide notes you want in git

## Templates you might want

Issue template for problem sets

Writeup template

## Notes

Conventions

1. Each class repo has a repo folder and a local folder.
2. Do not store large data or private exports in git.
3. Keep notes in Notion, and link them from each class README.
